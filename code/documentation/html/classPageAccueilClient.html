Detailed Description

La classe PageAccueilClient correspond à la vue de la page d'accueil pour les Client.

L'interface graghique comporte un QLabel contenant "Bonjour," suivit du prénom et nom du Client connecté. Elle contient également deux QPushButton : un permettant de créer un nouveau ticket et l'autre permettant d'accéder à l'historique des tickets du Client.
